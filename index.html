Dúvidas?

Perguntas Frequentes

Quanto tempo até ver resultados?

+
Sistema funcionando: imediatamente após onboarding. Primeiros leads qualificados: 3-5 dias. Primeiras reuniões agendadas: 1-2 semanas. ROI positivo: tipicamente 30-60 dias.

Preciso de conhecimento técnico?

+
Não para usar o Prospex AI ou Surge - são plug-and-play. Para o Setup, é bom ter alguém minimamente técnico na equipe para operar depois, mas treinamos completamente.

Funciona para qualquer nicho B2B?

+
Sim, desde que seja B2B. Personalizamos os prompts e fontes de dados para seu mercado específico. Quanto mais definido o ICP, melhores os resultados.

Qual a diferença entre Prospex AI, Setup e Surge?

+
Prospex AI: Plataforma SaaS onde você opera. R$ 297-997/mês.
Prospex Setup: Implementamos tudo na sua infraestrutura. R$ 4.997-14.997 (único).
Prospex Surge: Nós operamos e entregamos leads prontos. R$ 2.497-7.997/mês.

Os leads são exclusivos?

+
Sim, no Prospex Surge os leads são 100% exclusivos - não compartilhamos com outros clientes. No Prospex AI, você mesmo faz a coleta, então são naturalmente exclusivos.

Como funciona o SDR com IA?

+
Nosso SDR IA usa metodologia SPIN Selling para qualificar leads via WhatsApp. Ele faz perguntas de Situação, Problema, Implicação e Need-payoff, identifica oportunidades reais, trata objeções e agenda reuniões automaticamente no seu calendário.

Integra com meu CRM?

+
Sim! Temos integração nativa com Pipedrive, HubSpot e RD Station. Outros CRMs podem ser conectados via Zapier, Make ou API direta.
